## Clean
```shell
npm run clean
```
Project clean.

## Setup
```shell
npm run setup
```
Project setup.

## Build
```shell
npm run build
```
Builds the project.

## Unit tests
```shell
npm run test.unit
```
The code coverage report will be displayed in the terminal.

## Integration tests
```shell
npm run test.integration
```
These tests check that the package can be imported without errors in a new project.

## All tests
```shell
npm run test
```
Runs unit tests and integration tests.
