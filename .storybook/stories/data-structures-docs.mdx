# ca-data-structures

> `@ca-webstack/data-structures` — Core data structure utilities for TypeScript.

## Overview

Provides fundamental data structures missing from the JavaScript standard library: `List<T>`, `Dictionary<TKey, TValue>`, `DateOnly`, `DateTime`, `DateTimeOffset`, `DateRange`, `TimeSpan`, and a `Ref<T>` helper for pass-by-reference semantics.

## Installation

```bash
npm install @ca-webstack/data-structures
```

### Peer Dependencies

- `tslib`

## Quick Start

```typescript
import { List, Dictionary, DateTime, TimeSpan } from '@ca-webstack/data-structures';

const list = new List<string>();
list.add('hello');
list.addRange(['world', '!']);
console.log(list.count); // 3

const dict = new Dictionary<string, number>();
dict.set('a', 1);
dict.set('b', 2);
console.log(dict.containsKey('a')); // true
```

## Key Exports

| Class / Type | Description |
|---|---|
| `List<T>` | Strongly typed list with index access, insert, remove, sort, reverse. |
| `Dictionary<TKey, TValue>` | Key-value collection with add, get, set, remove, containsKey/Value. |
| `DateOnly` | `Date` subclass representing a date without time. |
| `DateTime` | Composite date + time value with multiple constructor overloads. |
| `DateTimeOffset` | Date-time with timezone offset; parses/produces ISO 8601 strings. |
| `DateRange` | Start/end date pair. |
| `TimeSpan` | Time interval with days/hours/minutes/seconds/ms; arithmetic and parsing. |
| `Ref<T>` / `IRef<T>` | Pass-by-reference wrapper (ByVal, ByRef model+property, ByRef getter/setter). |
