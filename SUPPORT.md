# Support

Thanks for using the CodeArchitects Enterprise Platform — WebStack! Here is how
to get help.

## Questions and usage help

- **Read the docs first** — start with the [README](README.md) and each
  package's own README.
- **Search existing issues** — your question may already be answered.
- **Open a discussion or issue** — if you can't find an answer, open a GitHub
  issue using the *Question* option and describe what you're trying to do.

## Reporting bugs and requesting features

Use the GitHub issue templates:

- **Bug report** — for something that isn't working as documented.
- **Feature request** — for a new capability or improvement.

Please provide a minimal reproduction, versions (Node, npm, the affected
`@ca-webstack/*` package), and expected vs. actual behavior.

## Security issues

Do **not** open a public issue for security problems. Follow the private
reporting process in [SECURITY.md](SECURITY.md).
