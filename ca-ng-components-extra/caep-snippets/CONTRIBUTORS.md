# Contributing to CAEP-snippets for Visual Studio Code

We welcome contributions and appreciate your help in making this extension even better.

## Found an Issue?

If you find a bug in the source code or documentation, please help us by submitting an issue to our Repository. Better yet, you can submit a Pull Request with a fix.

## Want a Feature?

You can request a new feature by submitting an issue to our Repository. If you'd like to implement a new feature, please submit an issue first with a proposal for your work to ensure that it aligns with the project's goals. Small features can be directly submitted as a Pull Request.

## Submitting an Issue

Before submitting an issue, please search the archive to see if your question has already been answered. If you find a bug that hasn't been reported, please provide the following information to increase the chances of a quick resolution:

- Overview of the issue, including any error messages
- Version affected
- Explanation of what you're trying to do and why the current behavior is a bug
- Steps to reproduce the error
- Any related issues
- Suggestions for a fix (if you have one)

Thank you for your contribution!
