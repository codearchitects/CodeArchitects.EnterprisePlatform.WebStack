# CAEP-snippets

A Visual Studio Code extension that provides CAEP snippets for TypeScript, HTML, yaml and scss.

## Features

- Collection of code templates for CAEP in TypeScript and HTML
- Easily accessible through IntelliSense and the dedicated snippet picker in the Command Palette
- Tab-completion for easy insertion of snippets

## Installation

We provide you the VSIX file, you can install the extension with:

```
 code --install-extension caep-snippets-0.0.1.vsix
```

## Usage

Type the trigger text and press Tab to insert the desired snippet.
Access the snippets through IntelliSense or the dedicated snippet picker in the Command Palette.
